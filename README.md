# 📌 JoyRecord Server
JoyRecord 서버는 **Spring Boot** 기반으로, MySQL을 활용한 데이터 관리 및 JWT 인증 시스템을 사용했습니다. 

<br>

## 🛠️ 기술 스택  
- **Backend:** Spring Boot, Spring Security, Mybatis  
- **Database:** MySQL  
- **Auth:** JWT  

<br>


## 📑 주요 기능  
- ✅ 사용자 **회원가입 및 로그인** (JWT 인증)  
- ✅ 영화, 드라마, 게임, 스포츠 경기 **기록 및 조회**  
- ✅ **관리자 기능** (회원 관리, 취미 관리, 기록 데이터 관리)
